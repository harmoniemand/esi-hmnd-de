A LORA-based sensor network for agriculture

ESI.env is a LORA-based sensor network designed for wineyards. We try to provide a backbone to collect data like temperature, humidity or rain. The projekt was started for the MILSET ESI Science Faire in Abu Dhabi. Working together with Albrecht Kiessling Wineyard in Heilbronn, Germany.

In germany we are facing a massiv shortage on water, especialy for farming purposes. Ti use the available water more efficient, it is neccecary to know where water is needed and to water the wineyards more selective. To do so we are collecting environmental data to analyse an show them on a map. This maps can be used by farmers to optimize water usage.

The sensors send their data using LoraWAN - a ultra low power radio transmission. We will use solar panels to power the sensor stations.

As the project is still under heavy development, we have build some test sensors to get some data to work with. But we do not have real statistics or analytics at this time.

LORA
LoRa (Long Range) is a spread spectrum modulation technique derived from chirp spread spectrum (CSS) technology and is the first low-cost implementation of chirp spread spectrum for commercial usage. It was developed by Cycleo of Grenoble, France, and acquired by Semtech in 2012, a founding member of the LoRa Alliance.
